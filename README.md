### **Adding Custom Playbooks**

Create `playbooks/custom.json`:
```json
{
    "name": "My Custom Analysis",
    "description": "Custom workflow for specific challenges",
    "category": "custom",
    "file_types": [".custom"],
    "execution_strategy": "sequential",
    "methods": [
        {
            "name": "Custom Tool",
            "description": "Run my custom tool",
            "type": "tool",
            "tool": "my-tool",
            "args": ["{target}"],
            "timeout": 60
        }
    ]
}
```
